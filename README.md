# ReadCut Releases

Official Early Access builds and installers for ReadCut.  https://www.readcut.xyz

## About ReadCut

ReadCut is a local AI-powered audio editing tool that makes editing audio as easy as reading text.

It transcribes speech, separates speakers, aligns timestamps, and turns long recordings into an editable script. You can cut audio by editing text, refine content directly in the transcript, regenerate parts with speaker timbre, and export both audio and transcript.

ReadCut introduces a text-first workflow for audio editing, making podcast and spoken-content production faster, clearer, and more natural.
